# Agent Governance Examples

Runnable examples showing the unified governance stack in action.

## Examples

| Example | Description | Components Used |
|---------|-------------|-----------------|
| [quickstart.py](quickstart.py) | Boot the full governance stack in 30 lines | OS Kernel, AgentMesh |
| [governed_agent.py](governed_agent.py) | Complete governed agent with policy, trust, and monitoring | All 4 components |

## Prerequisites

- Python 3.10+
- Install dependencies from this directory:

```bash
pip install -r requirements.txt
```

## Running

From this directory (`agent-governance-python/agent-compliance/examples/`):

```bash
python quickstart.py
python governed_agent.py
```
