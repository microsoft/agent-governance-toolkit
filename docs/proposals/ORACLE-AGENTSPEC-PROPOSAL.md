# Proposal: Oracle Agent Spec Governance Controls Mapping

**Status:** 🔄 Active — Oracle team reviewing, discussion ongoing  
**Author:** Imran Siddique (Microsoft)  
**Created:** 2026-03-08  
**Target:** oracle/agent-spec

## Summary

Map AGT governance controls to the Oracle Agent Spec, enabling spec-level declarations for capabilities, trust requirements, and policy references.

## External Engagement

- Issue: [oracle/agent-spec#125](https://github.com/oracle/agent-spec/issues/125) — Collaboration proposal
- Oracle team (dhilloulinoracle) responded positively, evaluating alignment

## Proposed Alignment Areas

1. **Capability declarations** — Ring-based permission model (Ring 0-3) informing standard capability declarations
2. **Trust and identity requirements** — DID-based identity with trust verification requirements in agent specs
3. **Governance policy references** — Specs referencing governance policies agents must comply with
